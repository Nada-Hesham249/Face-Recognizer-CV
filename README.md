# Face Detection & Recognition System 

A computer vision project focused on **face detection, feature extraction, and recognition** using classical **PCA (Eigenfaces)** approach.

The system is evaluated using standard face datasets and performance metrics including ROC curves.

---

## ✨ Features


### 🎯 Face Detection Engine
- Detects faces in grayscale and color images
- Preprocessing for illumination normalization
- Bounding box visualization around detected faces

### 🧠 PCA + SVM Face Recognition Pipeline

- Applies **PCA** for feature extraction and dimensionality reduction of face images  
- Converts each face into a compact feature vector in PCA space  
- Trains an **SVM classifier** on the extracted PCA features for recognition  
- Uses SVM decision boundaries to classify and identify face identities

### 📈 Model Evaluation

- Plots **ROC Curve** to evaluate classification performance  
  - True Positive Rate (TPR)  
  - False Positive Rate (FPR)  

- Computes **Accuracy score** for overall recognition performance  
- Visualizes ROC curve for threshold analysis and model validation
